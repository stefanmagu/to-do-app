# React + Vite

A simple To do app

# How to run the app

In the terminal, run the following commands:

1. **Install dependencies:**
   ```sh
   npm install
   ```

2. **Start the development server:**
   ```sh
   npm run dev
   ```

**Optional**

3. **Build for production:**
   ```sh
   npm run build
   ```

4. **Preview the production build:**
   ```sh
   npm run serve
   ```
